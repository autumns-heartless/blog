WebStackPage 个人导航站 ​

源码：GitHub 。

动因 ​

浏览器书签太多了，想把其中大部分公共站点都放到导航站，提高自己浏览器书签的信息密度，方便查找有用的书签。

大致找了一下个人导航站开源项目，最后决定使用 WebStackPage，手撕 HTML。

开发环境 ​

在 WebStackPage 下载压缩包，然后解压到本地，用 VS Code 打开。

VS Code 使用 Live Server 插件，方便在线调试。

为什么不 git clone

之所以下载压缩包，而不是 git clone，是为了保留一份源文件，方便参照。

当然也可以 clone 两份，不过我还是习惯下载压缩包。
